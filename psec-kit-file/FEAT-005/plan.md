# Plan: Organizer Dashboard (FEAT-005)

## الخطوات التنفيذية
1. **تصميم قاعدة البيانات:**
   - إضافة جدول الفعاليات (Events) مع ربطه بحساب المنظم (OrganizerId).
2. **تطوير واجهة المستخدم (Angular):**
   - صفحة لوحة تحكم المنظم (Organizer Dashboard) تعرض قائمة الفعاليات الخاصة بالمنظم.
   - نماذج (Forms) لإضافة وتعديل الفعاليات.
   - زر حذف مع تأكيد.
3. **تطوير واجهات برمجة التطبيقات (ABP/C#):**
   - Endpoints لإدارة CRUD للفعاليات (Create, Read, Update, Delete) مع تحقق من ملكية المنظم.
   - تحقق من صحة البيانات (Validation) في الخادم والعميل.
   - تطبيق القيود (لا تعديل/حذف بعد قبول الإدارة).
4. **ربط الواجهة بالخلفية:**
   - استدعاء API من Angular وعرض النتائج وتحديثها مباشرة.
5. **اختبار الميزة:**
   - اختبارات وحدات (Unit Tests) للمنطق.
   - اختبارات تكامل (Integration Tests) للواجهات.
6. **تعريب كامل للوحة التحكم.**

## الموارد المطلوبة
- مطور واجهات أمامية (Angular)
- مطور خلفية (C# / ABP)
- مختبر جودة
