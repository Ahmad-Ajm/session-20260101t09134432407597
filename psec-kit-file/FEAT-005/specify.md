# Specification: Organizer Dashboard (FEAT-005)

## المتطلبات الوظيفية
1. **إضافة فعالية:**
   - نموذج يحتوي الحقول: اسم الفعالية، التاريخ، الوقت، المدينة، المكان، التصنيف، وصف مختصر، الجهة المنظمة (يملأ تلقائياً).
   - تحقق من صحة البيانات (جميع الحقول مطلوبة).
2. **تعديل فعالية:**
   - إمكانية تعديل الفعالية قبل قبول الإدارة فقط.
   - نفس نموذج الإضافة مع تحميل بيانات الفعالية.
3. **حذف فعالية:**
   - زر حذف مع نافذة تأكيد.
   - الحذف متاح فقط قبل القبول الإداري.
4. **استعراض الفعاليات:**
   - جدول يعرض جميع فعاليات المنظم مع حالة كل فعالية (قيد المراجعة، مقبولة، مرفوضة).
5. **القيود:**
   - لا يمكن تعديل/حذف فعالية بعد قبولها إلا بطلب خاص (خارج المنصة).
   - كل فعالية مرتبطة بحساب منظم واحد فقط.
   - لا دعم لفرق التنظيم أو نقل ملكية الفعالية.
6. **اللغة:**
   - جميع الحقول والنصوص باللغة العربية.

## المتطلبات غير الوظيفية
- واجهة متوافقة مع الهواتف المحمولة (Responsive).
- تحقق من صحة البيانات على الواجهة والخادم.
- تجربة مستخدم بسيطة وسريعة.

## واجهات برمجة التطبيقات (API)
- GET /api/organizer/events
- POST /api/organizer/events
- PUT /api/organizer/events/{id}
- DELETE /api/organizer/events/{id}
