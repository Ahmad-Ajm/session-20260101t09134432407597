# Tasks for FEAT-007: Admin Control Panel

1. إعداد صلاحيات Admin في النظام (backend & frontend).
2. إنشاء route ولوحة تحكم إدارية في Angular.
3. بناء واجهة مراجعة الفعاليات (جدول - تفاصيل - أزرار قبول/رفض/تعديل).
4. بناء واجهة إحصائيات الفعاليات (جداول/رسوم بيانية).
5. بناء واجهة سجل العمليات (audit log).
6. تطوير API endpoints:
   - جلب الفعاليات بانتظار الموافقة
   - قبول فعالية
   - رفض فعالية
   - جلب الإحصائيات
   - جلب سجل العمليات
7. ربط الواجهات مع الـ API.
8. حماية جميع العمليات بنظام الصلاحيات.
9. إضافة تأكيد قبل تنفيذ أي إجراء حساس (modal confirmation).
10. اختبار الصلاحيات والوظائف (unit & integration tests).
11. مراجعة الأداء وتحسين التصفية/الترقيم في الواجهات.
12. توثيق جميع الوظائف والواجهات.
