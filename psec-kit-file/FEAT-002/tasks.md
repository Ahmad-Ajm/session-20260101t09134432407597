# Tasks for FEAT-002: Events Listing Interface

1. إعداد نموذج بيانات الفعالية في الواجهة (Angular) والخلفية (C# ABP).
2. بناء API لاسترجاع قائمة الفعاليات (GET /api/events?status=upcoming).
3. تطوير مكون Angular لعرض قائمة الفعاليات (Cards/Table).
4. دعم التصفح (Pagination أو Infinite Scroll) في الواجهة.
5. ربط المكون مع API واستعراض البيانات.
6. دعم اللغة العربية (اتجاه النص، المحاذاة).
7. دعم الاستجابة للأجهزة المحمولة.
8. اختبار الأداء مع بيانات كبيرة.
9. معالجة حالة عدم وجود فعاليات (رسالة مناسبة).
10. مراجعة التصميم النهائي مع عينات بيانات حقيقية.
